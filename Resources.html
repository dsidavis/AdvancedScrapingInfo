    Different Browsers

    Different Web browsers do slightly different things with dynamic content (and non-dynamic) and can make some information more readily available. Hence it is useful to look at documents using Google Chrome, Firefox, Safari, Opera, etc.

    There are also extensions (e.g. greasemonkey) that allow one to inject JavaScript code after a page is downloaded, displayed, etc.

    Web Browser Developer Tools

    These are useful for exploring HTML code, HTML elements within a document and how to specify XPath expressions/CSS selectors to extract them, see details of HTTP queries, find cookies, interact with JavaScript variables and code.
